# 12-Factor App 발표 자료

현대적 클라우드 네이티브 애플리케이션 개발 원칙에 대한 내부 개발자 교육용 발표 자료입니다.

## 🚀 라이브 데모

[GitHub Pages에서 보기](https://songminseok.github.io/learn-12-factors-app)

## 📋 내용

- 12-Factor App 개요
- 각 Factor별 설명 및 사례
- CSCF(Cloud Service Configuration Framework)와의 연관성
- 실제 적용 사례

## 🛠 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 미리보기
npm run preview
```
